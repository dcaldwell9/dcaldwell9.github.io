Data Encoding Table

Component Type Number of bits/range Description
Starting Location ASCII 264bits (0-32 characters) This where I started on my first day of high school. I started at my house which should be about 32 characters for my address.
Time when I left my house ASCII 40 bits (0-5 characters) The time at which I left my house to get onto the bus. It only needs up to 5 characters.
Time which I reached my destination ASCII 40 bits (0-5 characters) The time that I got to school after getting off of the bus. The time only needs 5 characters.
Destination ASCII 320 bits (0-40 characters) This where I got off of the last bus at. This is my school where I started high school. 40 characters is enough for the address of my school.
End ASCII 264 bits (0-32 characters) This is the buses that I got on. I got on two different buses which went different ways which should be about 32 characters.
